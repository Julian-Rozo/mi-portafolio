Mis proyectos

Convertidor de moneda

En este proyecto practiqué la conexión a una API para lograr traer en tiempo real los valores de las distintas monedas y con JavaScript agregar la funcionalidad y lógica matemática para hacer la conversión y obtener el resultado en la moneda que desees.

Desencriptador de texto

Este proyecto hizo parte de mi formación en ONE y con JavaScript trabajé la lógica para recibir un texto y poder encriptarlo reemplazando cada vocal por una cadena de texto aleatoria y así enviar un código secreto a mis amigos... también puedes desencriptar textos codeados.

Míralo en Vercel

Adivina el número

Como parte de mi formación en ONE trabajé este proyecto, se trata de un "adivina el número", allí la app determina un número random y te pide ingresar el primer intento, te dirá si es mayor o menor y cuando logres adivinarlo sabrás cuántos intentos te tomó. Lo mejor de todo es que puedes reiniciar y volver a jugar.

Alura Play

En este simil de youtube inicialmente consumí una API local con un archivo .JSON, posteriormente use My Json Server, que servicio online tan interesante. Acá puedes agregar videos alimentando la API y actulizando la página inicial, también podrás usar la barra de búsqueda para filtrar los resultados... es la primera vez que hago un filtro.
